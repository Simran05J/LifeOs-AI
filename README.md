# 🚀 LifeOS AI

> **Your Ambient Multi-Agent Personal AI Assistant**

LifeOS AI is a production-ready **Multi-Agent AI Assistant** built as part of the **Google × Kaggle AI Agents Capstone Project**.

Unlike traditional AI chatbots, LifeOS AI uses multiple specialized AI agents coordinated by an **Antigravity Orchestrator** to understand user requests, collaborate across domains, and provide intelligent responses through text and voice.

# 🤖 Multi-Agent Collaboration

LifeOS AI is capable of coordinating multiple AI agents automatically.

Example:

User:

> Plan my Goa trip under ₹20,000 and remind me tomorrow to book train tickets.

The system automatically coordinates:

- Travel Agent
- Reminder Agent

Another example:

User:

> I'm stressed because of exams. Plan my weekend and remind me to meditate every morning.

The system automatically coordinates:

- Wellness Agent
- Planner Agent
- Reminder Agent
